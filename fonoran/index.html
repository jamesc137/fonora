Fonoran

An experimental language and language-building toolkit.

Fonoran is an open-source experiment in language creation built on top of the Fonora phonetic writing system.

Rather than focusing on a single constructed language, the project explores how languages can be intentionally designed, analyzed, and evolved using modern tooling. It includes systems for creating primitive concepts, building compound words, exploring semantic relationships, reviewing vocabulary, and tracing the evolution of every word.

Fonoran serves as the reference language and a living proof of concept used to develop and validate the language building platform.

Fonora vs. Fonoran

Fonora

The writing system

Fonora is the phonetic writing system. It defines how speech is represented using a small set of phonetic symbols.

  • A universal phonetic script.
Explore the script

Fonoran

The language

Fonoran is the reference language built with that writing system and the accompanying language-builder toolkit.

  • A living language experiment and proof of concept for language creation.

The long-term goal is not simply to build one language, but to explore better tools for creating, understanding, and documenting languages using a shared phonetic foundation.
