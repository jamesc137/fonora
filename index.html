Fonora

A Phonetic Oral Script Language

Fonora is an experimental open-source writing system designed to represent spoken language using a small set of intuitive symbols.

Unlike traditional alphabets, each symbol is based on how a sound is physically produced in the mouth, helping readers understand both how a sound is written and how it is spoken.

What is Fonora?

Fonora is an open-source project created by James Calhoun.

The goal is to develop a universal phonetic writing system built from only nine core symbols.

These symbols represent locations and actions within the human vocal system, allowing written text to visually describe how sounds are produced.

Rather than memorizing arbitrary letters, readers learn a small set of symbols that correspond to physical speech mechanics.

Core Principles

Universal

Fonora is designed to represent spoken sounds rather than English spelling rules, allowing words from many languages to be written using the same system.

Phonetic

Words are written according to pronunciation rather than traditional spelling. For example, words that sound identical may share the same Fonora spelling regardless of their conventional spelling.

Oral

The symbols are based on where and how sounds are created within the mouth and vocal tract.

Minimal

The system is intentionally constrained to a small number of visual primitives to maximize learnability and simplicity.

Open Source

Fonora is an experimental community project that can evolve through research, testing, and contribution.

Project Goals

  • Create a highly learnable writing system
  • Minimize memorization
  • Encode pronunciation directly
  • Work across multiple languages
  • Remain visually simple
  • Stay open and accessible

Experimental Research Project

Fonora is currently an active research and development project.

The language, symbols, encoding rules, and tooling may evolve as testing and experimentation continue.
