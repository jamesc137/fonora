Fonoran

Communicate by constructing meaning, not memorizing it.

Fonoran is an experimental language built on Fonora Script to test whether people from different native languages can communicate by combining a small shared set of semantic roots.

Rather than memorizing thousands of unrelated words, speakers build meaning from simple concepts. If two people know the same roots, they should be able to invent expressions that the other person can reasonably understand, even if they don't choose exactly the same wording.

Every root, compound, and grammar rule is evaluated by a single question:

If someone only knew the shared roots, would they probably recover the intended meaning?

The goal is communication, not perfect decomposition.

Our central hypothesis is simple:

Can a language designed from first principles be learned quickly enough that two people with no shared native language can achieve practical communication after only a short period of study?

A Tiered Language

Fonoran is intentionally built in layers.

Communicative Core (~50 roots)

The minimum vocabulary needed to communicate everyday survival and social ideas.

Every proposed root must pass the Campfire Test:

Could this realistically come up between two strangers with no shared language during their first week together?

Extended Core (~100 roots)

Expands communication into everyday conversation while preserving the same compositional approach.

Complete Language

An open-ended vocabulary for specialized, technical, scientific, and abstract concepts.

Roots are introduced in the order humans experience the world:

survival → body → space → objects → people → actions → emotions → time → thinking → abstraction

Abstract concepts are added only when simpler combinations consistently prove insufficient.
